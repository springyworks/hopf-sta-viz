This runs the real Rust + wgpu + WGSL Spacetime-Algebra field simulation (the same code as the native app) compiled to WebAssembly.

It needs a WebGPU-capable browser — Chrome/Edge 113+ or Safari 18+ (Firefox: enable dom.webgpu.enabled). Compute shaders are not available on WebGL, so an older browser will stay on this screen.
