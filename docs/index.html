This runs Rust + wgpu compiled to WebAssembly. The GPU draws the hopfion particle field through a WebGL2 context while the CPU advects the cloud along the analytic Poynting vector.

Needs a WebGL2-capable browser (essentially every current desktop browser). The full compute-shader FDTD solver — which requires WebGPU — runs in the native app and in WebGPU browsers.
